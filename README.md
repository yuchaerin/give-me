# 🚀 Deployment (Vercel)

이 프로젝트는 **Vercel 자동 배포**를 사용합니다.

### 1️⃣ GitHub push

```
git add .
git commit -m "update hamster"
git push origin main
```

### 2️⃣ 자동 배포

```
GitHub push
↓
Vercel 감지
↓
자동 Build
↓
Deploy 완료
```

보통 **30초~1분** 정도 걸립니다.

# 🔧 Open Graph Preview

SNS 공유 시 아래 미리보기 카드가 표시됩니다.

```
질문 하나만 할게요
진지하게 대답해주세요 🙏
```

설정 코드

```html
<meta property="og:title" content="질문 하나만 할게요">
<meta property="og:description" content="진지하게 대답해주세요 🙏">
<meta property="og:image" content="https://give-me-six.vercel.app/preview.jpg">
<meta property="og:url" content="https://give-me-six.vercel.app">
<meta property="og:type" content="website">
```
